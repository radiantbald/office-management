# Floor Plan Vectorizer

Минимальный веб-сайт для превращения изображения плана этажа в SVG.

## Как запустить

1. Откройте `index.html` в браузере.
2. Загрузите план и нажмите "Векторизовать".

Если браузер блокирует доступ к локальным файлам, запустите простой сервер:

```
python3 -m http.server 8080
```

и откройте `http://localhost:8080` в браузере.

## Используемая библиотека

Векторизация выполняется через `imagetracerjs` (CDN).
